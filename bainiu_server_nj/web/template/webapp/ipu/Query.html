关于我们

亚信艾扑专注于智能终端开发框架与设计标准，以亚信的全体员工为受众群体，为智能终端项目、智能终端技术学习提供服务，未来将打造成亚信针对智能终端的开源共享服务平台。艾扑为使用者提供了简单易用、灵活支撑、贴合实际、封装完善的智能终端开发框架及其组件，使用者可参照使用说明快速的搭建起自己的工程项目，为项目的开发带来便捷的开发体验。艾扑还为技术人员提供智能终端开发与设计过程中的学习支持，为学习者提供源代码，使技术人员能够近距离的学习开发知识与设计标准。艾扑作为开源的平台，提供开源社区似的代码管理方式，任何人都可以共享自己的代码和提交框架的缺陷，以开源的贡献精神为平台的发展和框架的进步作出贡献。
